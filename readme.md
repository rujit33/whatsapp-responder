# WhatsApp Chat Widget

Embed the WhatsApp Chat Widget on your website using the following `<iframe>`.

## Embed Code

```html
<iframe
  src="https://rujit33.github.io/whatsapp-responder/?phone=xxxxxxxxxxxx"
  width="380"
  height="500"
  style="border: none"
  title="WhatsApp Chat Widget">
</iframe>
```

## Configuration

Replace:

- `xxxxxxxxxxxx` → Your WhatsApp phone number in **international format** (country code included, no `+`, spaces, or dashes).

### Example

If your WhatsApp number is:

```text
+977 9812345678
```

Use:

```html
<iframe
  src="https://rujit33.github.io/whatsapp-responder/?phone=9779812345678"
  width="380"
  height="500"
  style="border: none"
  title="WhatsApp Chat Widget">
</iframe>
```

## Notes

- Use the full international phone number.
- Do not include the `+` sign.
- Do not include spaces, hyphens, or parentheses.
- You can adjust the `width` and `height` values to fit your website layout.
